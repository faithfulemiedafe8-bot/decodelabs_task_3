# Project 3 - Organic Food Delivery App

## Overview

This project was completed as part of the DecodeLabs UI/UX Design Internship (Batch 2026).

The objective of this exercise was to design key screens for a mobile application that helps users browse and purchase fresh organic fruits and vegetables.

The project focused on creating a clean, trustworthy, and user-friendly experience while promoting healthy lifestyle choices through an intuitive mobile interface.

## Project Goal

The purpose of this project was to design a simple food delivery experience that allows users to discover products, view product details, and make purchasing decisions with minimal effort.

The design aimed to communicate freshness, health, and convenience while maintaining a modern and accessible user experience.

## What I Did

- Created a visual style guide for the application
- Designed a splash screen introducing the brand
- Designed a product listing screen for browsing available products
- Designed a product details screen with pricing and product information
- Established a consistent visual system using colors, typography, icons, and reusable components
- Focused on creating a simple and intuitive user journey

## Key Insight

One of the biggest takeaways from this project was the importance of consistency in mobile design. By creating a style guide first, it became easier to maintain visual consistency across multiple screens and create a more cohesive user experience.

## Why This Matters

Users should be able to quickly understand how an application works without feeling overwhelmed. Clear navigation, familiar design patterns, and consistent UI elements help build trust and improve usability.

This project demonstrated how design systems and screen design work together to create a better overall user experience.
